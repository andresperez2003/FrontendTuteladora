
  # Tuteladora Action Form

  This is a code bundle for Tuteladora Action Form. The original project is available at https://www.figma.com/design/KYFjD5zkHFRHlU5Q7UhpTb/Tuteladora-Action-Form.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Environment Variables

  For production deployment, configure the following environment variable in your hosting platform:

  - `VITE_API_URL`: Backend API URL (default: `https://generador-tuteladora.vercel.app`)

  ### Vercel Deployment

  1. Go to your project settings
  2. Navigate to "Environment Variables"
  3. Add `VITE_API_URL` with value `https://generador-tuteladora.vercel.app`
  4. Redeploy your application
